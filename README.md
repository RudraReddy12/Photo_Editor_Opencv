# Simple Photo Editor

A lightweight image editing tool built with OpenCV. Adjust dimensions, enhance visuals, apply filters, and transform your photos with ease

##  Features

- **Resize Images** - Adjust height and width precisely
- **Brightness & Contrast** - Fine-tune lighting and clarity
- **Edge Detection** - Highlight image edges with Canny algorithm
- **Blur** - Adjust the blur
- **Grayscale Conversion** - Convert to black & white
- **Image Rotation** - Rotate by any angle
